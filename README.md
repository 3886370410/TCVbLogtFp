# Python货物进销管理系统

## 核心功能
- 登录注册
- 采购订单管理
- 商品信息管理
- 供应商管理
- 销售管理
- 仓库管理
- 成员管理
- 个人信息修改

## 适用场景
- 企业内部货物进销管理
- 库存控制
- 销售数据分析
- 供应链管理

## 技术特点
- 使用Python Flask框架
- 前端技术：HTML、CSS、JavaScript、jQuery
- 后端技术：Python、Flask
- 数据库：MySQL
- 开发工具：Pycharm、Navicat
- 支持Python 3.0以上版本
- 支持MySQL 8.0版本
